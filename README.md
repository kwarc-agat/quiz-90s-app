# 90s pop culture quiz

Simple multimedia quiz app in theme of 90s pop culture. Loosely based on UX/UI original project.

## Features

- UX/UI project
- music questions
- video questions

## Technologies

- Android API 30
- Kotlin
- Adobe XD
